----------------------------------------------------------------

----------------------------------------------------------------
DESCRIPTION  (paste into Description field)
----------------------------------------------------------------
HomeBase replaces Chrome's default new tab with a calm, themed dashboard of the bookmarks you actually use, grouped into the categories that match how you think.

Most days you open a new tab and immediately type into the address bar — the new-tab page itself disappears. HomeBase makes the new-tab page worth landing on instead of skipping past.

Key features

- Curated bookmarks grouped into custom categories you control
- Five built-in themes plus a custom-color editor for background, cards, text, and accents
- Drag and drop to reorder links and move them between categories
- Quick search across titles, URLs, and category names
- JSON export and import for backups, machine migration, and version control
- Local-only: every change is saved instantly to your browser's storage
- Works offline. Your dashboard loads even when the internet doesn't
- Full keyboard navigation and screen-reader support

Privacy

Everything stays in your browser. No accounts, no servers, no telemetry, no analytics. The only permission requested is "storage", used to persist your bookmarks and theme selection.

Open source

GitHub: https://github.com/Mirabelle-Workspace/Built-By-Mirabelle
Built by Mirabelle Doiron — https://builtbymirabelle.com

SAVED LINKS  (new listing — first submission)

----------------------------------------------------------------
NAME
----------------------------------------------------------------
Saved Links


----------------------------------------------------------------
SUMMARY (132 char max)
----------------------------------------------------------------
Bulk-import open tabs into a categorized, searchable library. WCAG 2.1 AA, local-only, no account.


----------------------------------------------------------------
SINGLE PURPOSE
----------------------------------------------------------------
Save and categorize the URLs of your open Chrome tabs in a searchable local library.


----------------------------------------------------------------
DESCRIPTION  (paste into Description field)
----------------------------------------------------------------
Saved Links is the in-between layer for tab fatigue. One keyboard shortcut takes every tab in your current Chrome window and drops it into a categorized, tagged, searchable local library — so you can actually close those tabs without losing the thread.

Open-tab fatigue is real. You finish a research session with forty tabs across three windows and no good way to triage them. Bookmarking each one is friction. Closing them feels like loss. Reading-list apps want accounts. Saved Links is the in-between: tags, search, dedup, all local.

Key features

- One-click tab import. Press Alt+Shift+L (Option+Shift+L on Mac) and every tab in the current window lands in your library, deduped against what you already have, internal pages skipped automatically.
- Customizable categories. Rename or delete any category from its header. The seed is a starting suggestion, not a permanent structure.
- Tag filters. Click any tag pill at the top of the page to filter; click again to clear.
- Live search across titles, URLs, descriptions, tags, and category names.
- JSON export and import for backups and machine migration.
- Keyboard-first. Alt+Shift+S opens Saved Links, Esc closes the form, full Tab navigation everywhere.
- Dark mode and reduced-motion both respect your OS preferences.

Accessibility — WCAG 2.1 AA

Saved Links is audited and tested for WCAG 2.1 Level AA conformance. Every text and UI-component contrast pair is verified against the relative luminance formula. A 16-test Playwright and axe-core suite covers keyboard order, form error identification, reflow at 320 pixels, 200% zoom, prefers-reduced-motion, and accessibility-tree snapshots. Full audit:

https://github.com/Mirabelle-Workspace/Built-By-Mirabelle/blob/main/saved-links/AUDIT.md

Privacy

Local-only. Every link is stored in this extension's localStorage. Nothing leaves your browser. No accounts, no servers, no telemetry, no analytics, no cloud sync.

PERMISSION JUSTIFICATION — tabs
----------------------------------------------------------------
Required to read the URL and title of tabs in the current window when the user triggers a bulk import via the toolbar button or the Alt+Shift+L keyboard shortcut. The extension never reads tab data outside of these explicit user actions and never transmits the data anywhere.


----------------------------------------------------------------
PERMISSION JUSTIFICATION — scripting
----------------------------------------------------------------
Required so the background service worker can invoke the Saved Links page's in-page bulk-add API when the user presses the Alt+Shift+L keyboard shortcut. The script target is restricted to the Saved Links extension page itself; no external sites are scripted.


----------------------------------------------------------------
